Rift Watchtower

**Rift Watchtower** is a small [tower](/wiki/Skyrim:Imperial_Towers "Skyrim:Imperial Towers") east of [Ivarstead](/wiki/Skyrim:Ivarstead "Skyrim:Ivarstead") containing [Orc](/wiki/Skyrim:Orc "Skyrim:Orc") [bandits](/wiki/Skyrim:Bandit "Skyrim:Bandit").

## Walkthrough

A set of wooden ramps leads up to the door to the two-story watchtower. An Orc bandit guards the entrance, with another Orc bandit inside on the ground floor. The ground floor features a pile of firewood and a [tanning rack](/wiki/Skyrim:Tanning_Rack "Skyrim:Tanning Rack"), a table holding a mead barrel, a [leg of goat](/wiki/Skyrim:Leg_of_Goat "Skyrim:Leg of Goat"), two [leather strips](/wiki/Skyrim:Leather_Strips "Skyrim:Leather Strips"), two [garlic](/wiki/Skyrim:Garlic "Skyrim:Garlic") bulbs, a [rock warbler egg](/wiki/Skyrim:Rock_Warbler_Egg "Skyrim:Rock Warbler Egg"), two [raw rabbit legs](/wiki/Skyrim:Raw_Rabbit_Leg "Skyrim:Raw Rabbit Leg"), and a copy of *[The Code of Malacath](/wiki/Skyrim:The_Code_of_Malacath "Skyrim:The Code of Malacath")*, and an open fire and a cooking spit with a charred [skeever](/wiki/Skyrim:Skeever "Skyrim:Skeever") cooking over it which you can harvest for a [charred skeever hide](/wiki/Skyrim:Charred_Skeever_Hide "Skyrim:Charred Skeever Hide").

A stone spiral staircase leads to the top level, which is guarded by an Orc bandit and the Orc bandit chief. The top level features an Orc-styled shelter with two chairs and a long wooden table holding a joint of [raw beef](/wiki/Skyrim:Raw_Beef "Skyrim:Raw Beef"), [grilled leeks](/wiki/Skyrim:Grilled_Leeks "Skyrim:Grilled Leeks"), a bottle of [Nord mead](/wiki/Skyrim:Nord_Mead "Skyrim:Nord Mead"), an enchanted [Orcish warhammer](/wiki/Skyrim:Orcish_Warhammer "Skyrim:Orcish Warhammer"), a copy of the [Heavy Armor](/wiki/Skyrim:Heavy_Armor "Skyrim:Heavy Armor") [skill book](/wiki/Skyrim:Skill_Books "Skyrim:Skill Books") *[Hallgerd's Tale](/wiki/Skyrim:Hallgerd%27s_Tale "Skyrim:Hallgerd's Tale")*, and a copy of *[The True Nature of Orcs](/wiki/Skyrim:The_True_Nature_of_Orcs "Skyrim:The True Nature of Orcs")*. Behind the table is an unlocked chest, and to the left is a small table holding a random [bow](/wiki/Skyrim:Bow "Skyrim:Bow") and [arrows](/wiki/Skyrim:Arrows "Skyrim:Arrows").

## Notes

- There is an [orichalcum ore vein](/wiki/Skyrim:Orichalcum_Ore_Vein "Skyrim:Orichalcum Ore Vein") to the southeast.

## Bugs

- The Orc bandit chief does not respawn, which makes it impossible to complete [radiant quests](/wiki/Skyrim:Radiant "Skyrim:Radiant") at this location if he has been killed previously.
  - ![PC Only](//images.uesp.net/d/d7/Computer.svg "PC Only") The [Unofficial Skyrim Patch](/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch "Skyrim Mod:Unofficial Skyrim Patch"), version 2.0.1, fixes this bug.
  - [![On PC](//images.uesp.net/d/d7/Computer.svg)](https://en.wikipedia.org/wiki/PC_game "PC") Open the [console](/wiki/Skyrim:Console "Skyrim:Console"), then use the command `prid d1f56` followed by `resurrect`. This should revive the Orc and allow you to complete the quest as normal.

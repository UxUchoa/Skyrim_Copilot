# Skyrim:Dungeon Quests

## Overview

The following **side quests** and **miscellaneous quests** generally start when you approach or explore the listed **dungeons** and progress to completion within each dungeon. Each quest is categorized as a dungeon quest in the game data and the *[Prima Official Game Guide](/wiki/Books:The_Elder_Scrolls_V:_Skyrim:_Prima_Official_Game_Guide "Books:The Elder Scrolls V: Skyrim: Prima Official Game Guide")* and usually provides a backstory about the associated dungeon. For information on the many other quests involving dungeons, see the [Quests](/wiki/Skyrim:Quests "Skyrim:Quests") page.

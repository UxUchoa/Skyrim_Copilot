# Skyrim:Arcane Enchanters

## Overview

[![](//images.uesp.net/thumb/b/b9/SR-activity-Arcane_Enchanter.jpg/200px-SR-activity-Arcane_Enchanter.jpg)](/wiki/File:SR-activity-Arcane_Enchanter.jpg)

Arcane Enchanter

**Arcane Enchanters** are workbenches where [Enchanting](/wiki/Skyrim:Enchanting "Skyrim:Enchanting") can be done. New enchanted items can be created, or existing items can be broken down, allowing you to learn any new magical effects used in the item's enchantment. How an Arcane Enchanter is used is described on the [Enchanting page](/wiki/Skyrim:Enchanting#Skill_Usage "Skyrim:Enchanting"). A category listing places where enchanters can be found is available at [Skyrim-Places-Arcane Enchanters](/wiki/Category:Skyrim-Places-Arcane_Enchanters "Category:Skyrim-Places-Arcane Enchanters").

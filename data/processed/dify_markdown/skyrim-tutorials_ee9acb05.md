# Skyrim:Tutorials

## Overview

These **tutorial** quests provide a way to learn about the three crafting [skills](/wiki/Skyrim:Skills "Skyrim:Skills") ([Alchemy](/wiki/Skyrim:Alchemy "Skyrim:Alchemy"), [Enchanting](/wiki/Skyrim:Enchanting "Skyrim:Enchanting"), [Smithing](/wiki/Skyrim:Smithing "Skyrim:Smithing")). They are available from a few [NPCs](/wiki/Skyrim:NPCs "Skyrim:NPCs") in [Riverwood](/wiki/Skyrim:Riverwood "Skyrim:Riverwood"), [Whiterun](/wiki/Skyrim:Whiterun "Skyrim:Whiterun"), and [Falkreath](/wiki/Skyrim:Falkreath "Skyrim:Falkreath") near the game's starting point. The quest giver will show your character how to use the skills.

Note that you can use crafting skills without doing any of the tutorial quests, and the quests themselves do not give you any advantage in using the skills.

- **[Alchemy Tutorial](/wiki/Skyrim:Alchemy_Tutorial "Skyrim:Alchemy Tutorial")**: Learn the basics of potion making. ([radiant](/wiki/Skyrim:Radiant "Skyrim:Radiant"))
- **[Blacksmithing Tutorial](/wiki/Skyrim:Blacksmithing_Tutorial "Skyrim:Blacksmithing Tutorial")**: Learn the basics of [smithing](/wiki/Skyrim:Smithing "Skyrim:Smithing"). ([radiant](/wiki/Skyrim:Radiant "Skyrim:Radiant"))
- **[Enchanting Tutorial](/wiki/Skyrim:Enchanting_Tutorial "Skyrim:Enchanting Tutorial")**: Use an [Arcane Enchanter](/wiki/Skyrim:Arcane_Enchanter "Skyrim:Arcane Enchanter").
